Optional injectAsHtml

injectAsHtml: undefined | false | true
  • Defined in app/view/components/port.tsx:23

Use dangerouslySetInnerHTML to inject the element.

This is useful and only available when doing a server-side rendering. If element is static, turning on this option does not change how the page is rendered so much. There are two important changes to be considered when turning on this option:

  • Rendering using this option takes more processing power because entire the tree of element has to be serialized and deserialized again.

  • The rendered elements are distinct from element. Modifications made childrens of element are not visible and event handlers registered on them won't work.

This prop cannot be dynamically changed.
